## 最優先

0. **キャラクター名の権利・本人許諾の確認**。キャラクター名（罵尻ロマ子、BG、しんご、
   Hisa、たけちゃん、てるぞー、りおん、バク、さちよ、うえむ、みと、ピッコー、臥龍）は

   「見抜く快感」「読み合いの面白さ」といった体験面は未検証。docs/03_prototype_plan.md の
   手順を使い、実際のUIで最低5人×複数セッションを目安に実施する。
2. CPU AIの高度化。現状は「誇りが半分未満なら防御優先、それ以外は最大火力の攻撃」という
   単純なヒューリスティックのみ。プレイヤーの弱点推測体験とのバランスを見て、
   もう少し人間らしい振れ幅（時々あえて非最適な手を打つ等）を検討する。
