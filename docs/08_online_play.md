# オンライン2人対戦の起動方法

オンライン対戦は、6桁のルームコードを使うWebSocket方式です。プレイヤー1側がゲーム進行の正本を持ち、プレイヤー2には相手の手札を除いた状態だけを同期します。

## 本番起動

```bash
npm run build
npm start
```

既定では `PORT=8787` で、ビルド済み画面と `/ws` の対戦サーバーを同時に配信します。ホスティング先では `PORT` 環境変数が自動指定されても動作します。オンライン対戦を使う本番環境は、常駐Node.jsプロセスとWebSocket接続に対応している必要があります（静的ファイル配信だけではルーム通信は動きません）。

## Vite開発時

```bash
npm run dev
```

Vite開発サーバーにオンライン中継機能も組み込まれるため、別プロセスは不要です。開発画面は表示中のホストと同じ `/online-ws` に接続します。LAN内の別端末からも、開発PCのIPアドレスとViteのポートを指定すれば参加できます。

別ホストの中継サーバーへ接続する場合は、ビルド時に `VITE_WS_URL=wss://example.com/online-ws` を指定します。

## 対戦手順

1. タイトルの「2人対戦（オンライン）」を開く。
2. 片方が「新しいルームを作る」を押し、6桁コードを相手へ伝える。
3. 相手がコードを入力して参加する。
4. 双方がキャラクターを選び「準備完了」を押す。
5. プレイヤー1から交互に操作する。
